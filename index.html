About OfMaT
Welcome to OfMaT!
The OpenFlow Management Tool (OfMaT) is a web based tool to helping in monitoring and management of networks composed by OpenFlow devices. Ofmat acts in two ways: based in "Switch View", where it connects to all the swiches in the network and retrieve from them informations and statistics; and based in "Slice View", where it connects to the FlowVisor (a special purpose network controller) and gets information about existent virtualized networks (slicing).
Before you Start...
By default, Web Servers listen on port 80, but it is possible it is configured in another, so check it. If your network have a firewall, check that it is allowing traffic related to this web server. Finally, open the configuration file ("ofmat.conf") and make sure you had set correctly its parameters to have available all features from this tool.
