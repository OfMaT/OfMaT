About OfMaT
Welcome to OfMaT!
The OpenFlow Management Tool (OfMaT) is a web based tool to helping in monitoring and management of networks composed by OpenFlow devices. Ofmat was developed to support versions 1.0 and 1.3 of OpenFlow protocol. It has the "Switch View", where it connects to all the swiches in the network and retrieve from them informations and statistics; and it has "Slice View" (supporting only OpenFlow 1.0), where it connects to the "Network Virtualization", implemented in this release using FlowVisor (a special purpose network controller), and gets information about existent virtualized networks (slicing).
Before you Start...
If your network have a firewall, make the necessary configurations to allow acesses to this web server. Also open the configuration file ("ofmat.conf") and make sure you had set correctly its parameters to have available all features from this tool. The "Documentation" menu has detailed information about all configurations and resources from OfMaT, so we indicate you read it.
